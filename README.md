# Fledgewright site

Static marketing site for Fledgewright LLC, served via GitHub Pages.

- **Live branch:** `main`. The repo root of `main` is the published site.
- **Source of truth:** page copy, SEO metadata, and operational notes live in the
  studio's markdown files, not invented in HTML. The site renders from that copy so
  the two never drift.
- **Design variations:** built on their own branches, compared, then the chosen one
  is merged to `main`.
- **Custom domain:** fledgewright.com, pointed at Pages only at go-live.

`index.html` is a temporary placeholder and gets replaced by the built site.
